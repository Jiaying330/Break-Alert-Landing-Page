Break Alert

Break Alert is a Chrome Extension created for student studying at home and workers working at home. Break Alert includes a scheduler, a to-do list, an improved bookmark system and alarm to make your time spent on Chrome a productive one. For those looking to be extra productive, there is a built in productivity mode that will help you block out any distracting websites.

Break Alert is a free download that you can find on the Chrome store today. If you are interested in trying it out, download the extension using the "+ Add to Chrome" button above. If you are interested in the source code, click on the Github icon above.
